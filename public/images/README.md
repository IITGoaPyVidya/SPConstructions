## Photo Requirements
- **Format:** JPG, JPEG, or PNG
- **Size:** Recommended width 800-1200px for web display
- **Naming:** Use lowercase with hyphens (e.g., jcb-excavator-1.jpg)
- **Compression:** Compress images for faster loading (use tools like TinyPNG)
